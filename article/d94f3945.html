桌宠对话API使用文档

发表于2025-01-30|更新于2025-01-31|桌宠
|阅读量:

概述

本系统提供完整的智能语音交互服务，支持以下流程：

  1. 接收用户语音输入（MP3格式）
  2. 语音识别转文本（ASR）
  3. 结合RAG增强的大模型对话生成
  4. 文本转语音（TTS）生成回复
  5. 返回生成的语音文件路径

快速开始

接口文档

1. 语音对话处理

  • URL: /dialogue_tts/
  • 方法: POST
  • 认证: X-API-Key 请求头
  • 参数:
    • mp3_file (File): 用户上传的语音文件（MP3格式，16kHz/单声道）
    • reference_id (可选): 声音克隆参考音频ID
  • 响应示例:

2. 立即清理文件

  • URL: /clean_file/{filename}
  • 方法: DELETE
  • 参数:
    • filename: 要删除的文件名（如output_1a2b3c4d.mp3）

3. 上传知识文档

  • URL: /upload_RAG/
  • 方法: POST
  • 参数:
    • file (File): 支持格式：PDF/DOCX/TXT
  • 功能: 上传文档到RAG知识库，自动更新语义索引

4. 缓存状态查询

  • URL: /cache_status/
  • 方法: GET
  • 响应示例:

错误码参考

状态码 说明
401 无效的API密钥
400 无效请求/文件格式错误
403 非法文件路径操作
404 文件不存在
500 内部处理异常
503 第三方服务不可用

高级功能

RAG知识增强

  1. 支持格式:

    • PDF（图文内容提取）
    • DOCX（格式保留解析）
    • TXT（UTF-8编码）

  2. 检索机制:

    • 使用FAISS向量数据库
    • 相似度阈值：0.65（可配置）
    • 缓存最近100条查询结果

语音服务参数

服务类型 采样率 声道 最大时长
ASR输入 16kHz 单声道 60秒
TTS输出 24kHz 单声道 无限制

注意事项

  1. 文件管理:

    • 系统自动清理10分钟前的生成文件
    • 重要文件请及时通过/static/{filename}下载
    • 上传RAG文档单文件大小限制：50MB

  2. 性能优化:

    • 首次请求响应时间可能较长（向量库初始化）
    • 高频查询建议启用缓存
    • 推荐部署配置：4核CPU/8GB内存
